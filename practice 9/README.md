# PP2 Lab 9: Pygame Applications

This repository contains three Pygame applications developed as part of the Programing Principles II course.

## Projects Included:
1. **Mickey Clock**: A functional analog clock with Mickey Mouse hands.
2. **Music Player**: A digital player to manage and play MP3 tracks.
3. **Moving Ball**: A simple interactive program to control a ball within boundaries.

Ensure you have `pygame` installed:
